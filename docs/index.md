# Project Documentation Index

## Project Overview

- Type: monolith (backend API)
- Primary Language: Python
- Architecture: Django modular monolith

## Quick Reference

- Tech Stack: Django + DRF + PostgreSQL + Redis + Celery + Channels
- Entry Points: `manage.py`, `config/asgi.py`, `config/wsgi.py`
