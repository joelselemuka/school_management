## 🚀 Premiers Pas

### 1. Connexion

1. Ouvrez votre navigateur web
2. Accédez à `https://votre-ecole.com`
3. Cliquez sur **"Se connecter"**
4. Entrez vos identifiants:
   - **Nom d'utilisateur** ou **Email**
   - **Mot de passe**
5. Cliquez sur **"Connexion"**

### 2. Premier Login

Lors de votre première connexion:

1. Vous recevrez un email avec vos identifiants
2. **Changez immédiatement votre mot de passe**:
   - Cliquez sur votre nom (coin supérieur droit)
   - Sélectionnez **"Mon profil"**
   - Cliquez sur **"Changer le mot de passe"**
3. Remplissez vos informations personnelles

### 3. Mot de Passe Oublié

Si vous oubliez votre mot de passe:

1. Cliquez sur **"Mot de passe oublié?"**
2. Entrez votre email
3. Consultez votre boîte mail
4. Cliquez sur le lien de réinitialisation
5. Définissez un nouveau mot de passe

## 👨‍👩‍👦 Guide Parent

### Vue d'Ensemble

En tant que parent, vous pouvez:
- ✅ Consulter les notes de vos enfants
- ✅ Voir les présences/absences
- ✅ Payer les frais scolaires
- ✅ Communiquer avec l'école
- ✅ Télécharger les bulletins

### 🏠 Tableau de Bord

Après connexion, vous verrez:
- **Carte résumé** pour chaque enfant
- **Notifications récentes**
- **Prochains paiements**
- **Événements à venir**

### 📊 Consulter les Notes

1. **Navigation:**
   - Cliquez sur **"Académique"** > **"Notes"**
   - Sélectionnez l'enfant (si plusieurs)

2. **Filtres disponibles:**
   - Par matière
   - Par période/trimestre
   - Par type d'évaluation

3. **Informations affichées:**
   - Note obtenue
   - Note sur (max)
   - Moyenne de classe
   - Rang
   - Appréciation du professeur

### 📥 Télécharger un Bulletin

1. Allez dans **"Bulletins"**
2. Sélectionnez la période
3. Sélectionnez l'enfant
4. Cliquez sur **"Télécharger PDF"**
5. Le bulletin s'ouvre dans un nouvel onglet

**Format:** PDF, prêt pour impression

### 📅 Voir les Absences

1. Allez dans **"Présences"**
2. Vue calendrier avec code couleur:
   - 🟢 Vert = Présent
   - 🔴 Rouge = Absent
   - 🟡 Jaune = Retard

3. **Justifier une absence:**
   - Cliquez sur l'absence
   - Cliquez sur **"Justifier"**
   - Téléchargez le justificatif (photo/PDF)
   - Ajoutez un motif
   - Cliquez sur **"Envoyer"**

### 💰 Payer les Frais

1. **Voir le solde:**
   - Allez dans **"Finance"** > **"Mon compte"**
   - Vous verrez:
     - Solde actuel
     - Montant dû
     - Historique paiements

2. **Effectuer un paiement:**
   - Cliquez sur **"Nouveau paiement"**
   - Sélectionnez le frais à payer
   - Choisissez le mode de paiement:
     - Mobile Money (Airtel, Vodacom, Orange)
     - Virement bancaire
     - Espèces (à l'école)
   - Confirmez
   - **Téléchargez le reçu**

3. **Historique:**
   - Tous vos paiements sont archivés
   - Téléchargement des reçus possible

### 💬 Communiquer

1. **Messages:**
   - Cliquez sur l'icône **💬** (coin supérieur droit)
   - Nouveau message: Cliquez sur **"+"**
   - Sélectionnez le destinataire:
     - Professeur
     - Direction
     - Secrétariat
   - Tapez votre message
   - Envoyez

2. **Notifications:**
   - Cliquez sur l'icône **🔔**
   - Vous verrez:
     - Nouvelles notes publiées
     - Absences de l'enfant
     - Paiements enregistrés
     - Annonces importantes

### 📱 Application Mobile

**Disponible sur:**
- iOS (App Store) - *À venir*
- Android (Play Store) - *À venir*

**Fonctionnalités:**
- Notifications push
- Toutes les fonctions web
- Mode hors ligne (bulletins)

## 👨‍🏫 Guide Enseignant

### Vue d'Ensemble

En tant qu'enseignant, vous pouvez:
- ✅ Saisir les notes
- ✅ Prendre les présences
- ✅ Consulter votre horaire
- ✅ Communiquer avec parents/direction
- ✅ Planifier des évaluations

### 📚 Gérer mes Classes

1. **Voir mes classes:**
   - Allez dans **"Mes Classes"**
   - Vous verrez toutes vos classes assignées

2. **Consulter une classe:**
   - Cliquez sur le nom de la classe
   - Vous verrez:
     - Liste des élèves
     - Effectif
     - Statistiques

### 📝 Saisir les Notes

1. **Créer une évaluation:**
   - Allez dans **"Évaluations"**
   - Cliquez sur **"+ Nouvelle évaluation"**
   - Remplissez:
     - Nom (ex: "Interrogation 1")
     - Type (Interro, Devoir, Examen)
     - Date
     - Note sur (ex: 20)
     - Période/Trimestre
     - Classe
   - Cliquez sur **"Créer"**

2. **Saisir les notes:**
   - Cliquez sur l'évaluation
   - Cliquez sur **"Saisir les notes"**
   - **Option 1: Saisie manuelle**
     - Entrez la note pour chaque élève
     - Cliquez sur **"Enregistrer"**
   - **Option 2: Import Excel**
     - Téléchargez le modèle
     - Remplissez dans Excel
     - Importez le fichier
     - Vérifiez
     - Cliquez sur **"Valider"**

3. **Publier les notes:**
   - Par défaut, les notes sont **privées**
   - Pour les rendre visibles aux parents/élèves:
     - Cliquez sur l'évaluation
     - Cliquez sur **"Publier"**
     - Confirmez

⚠️ **Important:** Une fois publiées, les notes ne peuvent plus être modifiées!

### ✅ Prendre les Présences

1. **Méthode rapide (quotidienne):**
   - Allez dans **"Présences"**
   - Sélectionnez la classe
   - Sélectionnez le cours d'aujourd'hui
   - Cochez les **absents** et **retards**
   - Cliquez sur **"Enregistrer"**

2. **Par défaut:** Tous les élèves sont marqués présents

3. **Justification:**
   - Si un parent envoie un justificatif
   - Vous recevez une notification
   - Allez dans **"Justificatifs en attente"**
   - Validez ou rejetez

### 📅 Mon Horaire

1. Allez dans **"Mon Horaire"**
2. Vue hebdomadaire:
   - Lundi à Samedi
   - Heures de cours
   - Classes
   - Salles

3. **Export:**
   - Cliquez sur **"Exporter"**
   - Format: PDF ou iCalendar (pour Google Calendar)

### 📊 Statistiques de ma Classe

1. Allez dans **"Statistiques"**
2. Vous verrez:
   - Moyenne générale de la classe
   - Taux de réussite
   - Graphiques de distribution
   - Top 10 élèves
   - Élèves en difficulté

3. **Export pour réunion:**
   - Cliquez sur **"Exporter PDF"**

## ❓ FAQ (Questions Fréquentes)

### 🔐 Connexion & Sécurité

**Q: J'ai oublié mon mot de passe, que faire?**  
R: Cliquez sur "Mot de passe oublié" sur la page de connexion. Suivez les instructions envoyées par email.

**Q: Comment changer mon mot de passe?**  
R: Menu > Mon profil > Changer le mot de passe

**Q: Mon compte est bloqué, pourquoi?**  
R: Après 5 tentatives de connexion échouées. Contactez le support ou attendez 30 minutes.

### 📱 Technique

**Q: L'application fonctionne-t-elle sur mobile?**  
R: Oui, l'interface est responsive. Une app mobile native est en développement.

**Q: Puis-je utiliser plusieurs appareils?**  
R: Oui, connectez-vous depuis n'importe quel appareil.

**Q: Les données sont-elles sécurisées?**  
R: Oui, chiffrement HTTPS, sauvegardes quotidiennes, accès restreint par rôle.

### 📊 Notes & Bulletins

**Q: Quand les notes sont-elles publiées?**  
R: Les professeurs publient les notes après correction. Vous recevez une notification.

**Q: Puis-je contester une note?**  
R: Contactez d'abord le professeur via le chat. Si pas de solution, contactez la direction.

**Q: Où trouver mes anciens bulletins?**  
R: Dans "Bulletins", sélectionnez l'année académique puis la période.

### 💰 Finance

**Q: Quels modes de paiement sont acceptés?**  
R: Mobile Money (Airtel, Vodacom, Orange), Virement bancaire, Espèces à l'école.

**Q: Où voir mes reçus de paiement?**  
R: Finance > Historique des paiements > Cliquez sur un paiement > Télécharger reçu

**Q: Puis-je payer en plusieurs fois?**  
R: Oui, contactez le service financier pour établir un échéancier.

### 🔔 Notifications

**Q: Comment activer les notifications push?**  
R: Paramètres > Notifications > Activer

**Q: Je ne reçois pas les notifications, pourquoi?**  
R: Vérifiez:
- Paramètres de notification activés
- Email correct dans votre profil
- Dossier spam

## 📄 Glossaire

**Bulletin** - Document récapitulatif des résultats d'une période  
**Trimestre/Période** - Division de l'année académique (3 périodes)  
**Évaluation** - Interro, devoir, examen  
**Moyenne Générale** - Moyenne de toutes les matières  
**Rang** - Position de l'élève dans la classe  
**Frais Scolaires** - Minerval, matériel, uniforme, etc.  
**Dette** - Solde impayé  
**Justificatif** - Document prouvant une absence légitime
